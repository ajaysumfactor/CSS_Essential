## Selectors

#### Element Selector
#### Class Selector
    .nameOfclass{}
    Can applied on many element
#### Id selector
    #Id{}
    Apply on single element
#### Attribute Selector
    <p draggable='true'>Drag me </p>
        p[draggable]{
            color: red;
        }
#### Universal selector
    *{}
    select all
